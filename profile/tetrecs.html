TETRECS Multiplayer

A JavaFX tribute to the arcade classic that lets you practice solo runs or battle friends in real time with lobbies, live chat, and incoming garbage lines. Smooth controls, responsive UI, and a social layer make the game shine.

Gameplay Highlights

The single-player mode nails the fundamentals—precise rotation, quick-drop, and a satisfying scoreboard for chasing PBs. Multiplayer adds a live party lobby with chat, spectator support, and a shared queue so groups can launch matches instantly.

  • Built end-to-end in Java, leveraging JavaFX for the UI, animation loop, and responsive controller mapping.
  • Realtime networking layer keeps board states in sync while players trash-talk through the in-game chat.
  • Custom theming, onboarding tips, and control rebinding keep the experience friendly for newcomers.
